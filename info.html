Lightweight Information Page

Note: This page is designed to be extremely lightweight with minimal JavaScript. If you experienced performance issues with the chatbot, you can safely use this page.

About This Chatbot

This is a 100% client-side chatbot that runs entirely in your browser. No data is sent to any servers, ensuring complete privacy.

Features

  • Client-side processing - all conversations stay on your device
  • No tracking or data collection
  • Performance monitoring to prevent browser slowdown
  • Automatic message pruning to manage memory
  • Typing indicators and smooth animations

Performance Optimization

The chatbot includes several features to ensure smooth performance:

  • Message Limits: Automatically removes old messages to prevent memory issues
  • Idle Scheduling: Uses browser idle time for processing
  • Memory Monitoring: Alerts you if memory usage gets too high
  • Graceful Degradation: This info page as a fallback option

Frequently Asked Questions

Why might I experience performance issues?

Performance can be affected by:

  • Having many tabs open in your browser
  • Older or resource-constrained devices
  • Very long conversation histories
  • Other resource-intensive applications running

Is my data private?

Yes! Everything runs in your browser. No messages or data are sent to any server. The chatbot uses simple pattern matching to generate responses locally.

How can I improve performance?

  • Close unnecessary browser tabs
  • Restart your browser periodically
  • Use this lightweight info page instead of the chatbot
  • Clear your browser cache if needed
